Split Content

The Split Content pattern dictates a block of content where one portion (or column) is primary and takes up all available width, while secondary content is only as wide as it needs to be.

Spacing between column is usually responsive at $bp-md following the guidelines specified in the Spacing size chart.

Nucleus provides a base nc-split-content series of classes to apply this layout pattern.

nc-split-content Apply to container for the block
nc-split-content__primary Apply to the column that should grow to take up all available space
nc-split-content__secondary Apply to columns that only need to be as wide as needed for their content
nc-split-content__[primary|secondary]--top Use to modifiy any particular column to be top-aligned
nc-split-content__[primary|secondary]--bottom Use to modifiy any particular column to be bottom-aligned

Visually Hidden

In many situations, you will need to have content available for accessibility reasons, but do not want that content to actually be visible in the browser.

As an example, this "X" is commonly understood as a "close" icon. However, those who are visually impaired and use screen readers need to be able to also understand what this button does. You want descriptive text to be present but not visible.

<button class="nc-icon-button">
  <span class="nc-ada-text">Close</span>
  <svg class="nc-icon-close">
    <use xlink:href="img/sprite.svg#icon-plus"></use>
  </svg>
</button>

Nucleus provides the nc-ada-text class for this purpose, but you can also make your own class names by using the nc-visually-hidden() mixin.

Code Reference

With version 1.0, Nucleus now provides a mixin that allows you to customize the gutter between columns and whether this spacing should be responsive. The mixin will generate the child classes for you.

nc-split-content() Mixin mixins/_layout.scss

Generates series of classes for split-content layout pattern.

Parameters Default Description Required
$gutter moderate Set amount of spacing between columns (refer to Spacing)  
$responsive true Should the size be responsive   
/* basic usage */
.widget {
  @include nc-split-content('cozy', false);
}

/* will generate these class names:
    .widget {}
    .widget__primary {}
    .widget__secondary {}

    .widget__primary--top,
    .widget__secondary--top {}

    .widget__primary--bottom,
    .widget__secondary--bottom {}
*/

nc-visually-hidden() Mixin mixins/_layout.scss

Generates styles that hide content visually but allow it to remain accessible. This is a parameter-less mixin.

For more information about this method, please see this Jonathan Snook article.
