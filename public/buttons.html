Buttons

General specifications for buttons are as follows:

  • normal left and right padding
  • milli font size
  • 36px tall at less than 720px screen width
  • 38px tall at screen width 720px and above
  • Text is centered, bold, and all-caps

Code Reference

nc-btn() Mixin mixins/_btn.scss

Generates button styles based on parameters.

Parameters Default Description Required
$color nc-theme('primary-button-color') Button color  
$txtcolor $nc-white Button text color  
$height $nc-button-height Button height  
$hoverstate false Should there be a hoverstate for the button  
$outlined false Should this be the outlined button style  
$responsive true Should the button's height and width be responsive (font is always responsive)   
/* example usage */
.regular-button {
  @include nc-btn($hoverstate: true);
}

.non-responsive-button {
  @include nc-btn($hoverstate: true, $responsive: false);
}

.checkout-button {
  @include nc-btn(nc-theme('cta-button-color'), $hoverstate: true);
}

.outlined-button {
  @include nc-btn(nc-theme('primary-button-color'), $hoverstate: true, $outlined: true);
}

nc-unbutton() Mixin mixins/_btn.scss

Creates a button that appears as regular text (defaults link color).

Parameters Default Description Required
$txtcolor nc-theme('link-color') Text color of button  
$padding $nc-button-padding Left and right padding  
$nowrap nowrap Whether to allow button text to wrap or not  
$responsive true Should the button's dimensions be responsive   
/* create a link button whose test is colored like regular text */
.text-button {
  @include nc-unbutton(nc-theme('text-color'));
}

nc-icon-button() Mixin mixins/_btn.scss

Creates a button that consists only of an image. This mixin calls nc-button internally. Note that this mixin does not actually generate the icon itself; you'll need to provide that in the markup.

Parameters Default Description Required
None   This mixin accepts no parameters   
/* style a button to hold an image */
.an-icon-button {
  @include nc-icon-button();
}
