Themes

Nucleus has built-in support for Ticketmaster and Live Nation themes. These are located in the /themes directory, and are named tm.scss and ln.scss, respectively.

Nucleus is configured with the Ticketmaster theme by default. If you wish to use the Live Nation theme instead, specify it in the $nc-config map before you import Nucleus files.

/* LN theme */
$nc-config: (
  'theme': 'ln'
);

@import '[path/to/nucleus]/nucleus-core';

To apply theme colors in your styles, use the nc-theme function. You'll need to specify the keyname for the color you want to access.

/* Applying a theme color */
.thingamajig {
  color: nc-theme('alt-text-color');
  background-color: nc-theme('page-section-bg-color');
}

Customizations

If you need to customize the theme further (or if the existing key names are not applicable to your use case), you can create additional keys as a SASS map and merge them into the theme.

Nucleus maintains the active theme in a SASS map referenced by $nc-current-theme. After defining your additional keys, use the nc-theme-add function to add it to the theme.

Important Note

All of your color definitions should be abstracted in your theme definition. This is to support branding at the app level. (Themes are not always 1:1; the fact that everything that's blue in the TM theme does not necessarily mean that it is red in the LN theme.)

To be more clear, you shouldn't be using nc-color within any style definitions. You use the function to specify colors in your theme map, then apply those colors with nc-theme via an abstract key name.

Here is an example of adding your theme keys:

/* First bring in Nucleus's core  */
@import '[path/to/nucleus]/nucleus-core';

/* Define your custom theme keys to be added */
$your-theme: (
  'widget-title-color': nc-color('helen');
  'widget-link-color': nc-color('yosemite');
);

/* Add them to the theme */
$nc-current-theme: nc-theme-add($your-theme);

/* You can now access this key as you would with the existing keys in Nucleus */
.widget-title {
  color: nc-theme('widget-title-color');
}

Code Reference

nc-theme() Function themes/_themes.scss

Returns hex value from theme for a given key.

Parameters Default Description Required
$name   Name of theme key (e.g., link-color.) 
.widget {
  color: nc-theme('alt-text-color');
}

nc-theme-add() Function themes/_themes.scss

Adds a given SASS map of custom theme values to the working theme.

Parameters Default Description Required
$map   A SASS map 
/* Define your custom theme keys to be added */
$your-theme: (
  'widget-title-color': nc-color('helen');
  'widget-link-color': nc-color('yosemite');
);

/* Add them to the theme */
$nc-current-theme: nc-theme-add($your-theme);

nc-color() Function themes/_colors.scss

Returns hex value from palette.

Parameters Default Description Required
$name   Name of color. (e.g., sierra.)
$shade base Name of shade. (e.g., dark.)  
$map $nc-palette SASS map that contains palette.   
$theme: (
  'abstract-key-name': nc-color('malibu', 'muted');
);

nc-hover() Function themes/_colors.scss

Given a color, returns the hover state of that color.

Parameters Default Description Required
$color   Hex value. 
$darker: nc-hover(#19ab20);  /* #138018 */

nc-disabled() Function themes/_colors.scss

Given a color, returns the disabled state of that color.

Parameters Default Description Required
$color   Hex value. 
$darker: nc-disabled(#d0006f);  /* #f3bfdb */

nc-darker() Function themes/_colors.scss

Given a color, returns a new color mixed with a percentage of white (#000).

Parameters Default Description Required
$color   Hex value.
$percent 25% Percentage of white to mix in with the given $color.   
$darker: nc-darker(#19ab20);  /* #138018 */

nc-lighter() Function themes/_colors.scss

Given a color, returns a new color mixed with a percentage of white (#fff).

Parameters Default Description Required
$color   Hex value.
$percent 25% Percentage of white to mix in with the given $color.   
$lighter: nc-lighter(#f00, 50%);  /* #ff8080 */
