Labels

There are three label variations based on the background that the form field is sitting on.

Example Class Specifications
  • nc-input-group__label
milli, cascade, bold, slim bottom margin
  • nc-input-group__label
  • nc-input-group__label--dark
cascade (dark)
  • nc-input-group__label
  • nc-input-group__label--light
white

Text fields

<div class="input-group">
  <label for="email" class="input-group__label">
    Email <span class="input-group__required">*</span>
  </label>
  <input id="email" name="email" type="text" class="input-group__text-field">
</div>

Error State

Text fields in error state should add the --error modifier class to all Nucleus elements.

 
<div class="nc-input-group nc-input-group--error">
  <label for="email" class="nc-input-group__label nc-input-group__label--error">
    Email <span class="nc-input-group__required nc-input-group__required--error">*</span>
  </label>
  <input id="email" name="email" type="text" class="nc-input-group__text-field nc-input-group__text-field--error" placeholder="Email">
</div>

You can also use the nc-input-group__error-message class to show an error message.

This is a sample error message to show when something goes wrong with the input field.
