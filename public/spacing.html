Scale

Whitespace is an important part of creating rhythm and harmony between elements on a page. To maintain consistency, we've developed a spacing scale that has a 6px base size and a 1.5 ratio. All of our spacing values are derived as a multiple of these two numbers.

Code Reference

nc-margin() Mixin mixins/_spacing.scss

Generates margin values.

Parameters Default Description Required
$vals Set margins; can be either name from size chart, a list, or a SASS map
$responsive true Should the size be responsive   
/* basic usage */
.widget {
  @include nc-margin('compact');
}

/* fixed margins aka not responsive */
.text-fixed {
  @include nc-margin('moderate', $responsive: false);
}

If you need to specify non-uniform values, you can pass in a list of values. Note that the list must contain either 2 or 4 values and be defined in a clockwise direction. This is similar to how you would shorthand these values in native CSS.

/* these can be defined as a variable or passed inline within @include call */
$list2: ('normal', 'moderate');
$list4: ('cozy', 'cozy', 0, 'cozy');

.margin-shorthanded {
  @include nc-margin($list2);
}

A third option is to pass in a SASS map, which will allow you to specify any sides you wish and the order does not matter.

/* using a SASS map */
$map: (
  'top': 10px,
  'bottom': 0,
  'left': 'slim'
);

.margin-mapped {
  @include nc-margin($map);
}

nc-spacing-val() Function base/_helpers.scss

Returns spacing value for given name. This is useful for calculations or other situations where you only need the raw number and do not need the actual style definitions for setting the margin or padding.

Parameters Default Description Required
$val Name of spacing value (see size chart)
$set mini Which set to pull from; should rarely ever be anything other than the default
