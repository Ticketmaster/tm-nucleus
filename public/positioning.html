Positioning

Nucleus provides a set of mix-ins to easily apply a common set of positioning attributes.

Some examples of these in-action:

This is vertically centered
This is horizontally centered
This is dead center (vertical and horizontal)

Code Reference

nc-vertically-center() Mixin mixins/_positioning.scss

Vertically centers an element.

Parameters Default Description Required
$pos relative Either relative or absolute   
/* vertically center an element absolutely within its nearest relative positioning parent */
.vertical-center {
  @include nc-vertically-center(absolute)
}

nc-horizontally-center() Mixin mixins/_positioning.scss

Horizontally centers an element.

Parameters Default Description Required
$pos relative Either relative or absolute   
/* horizontally center an element absolutely within its nearest relative positioning parent */
.horizontal-center {
  @include nc-horizontally-center(absolute)
}

nc-dead-center() Mixin mixins/_positioning.scss

Centers an element both vertically and horizontally.

Parameters Default Description Required
$pos relative Either relative or absolute   
/* dead center an element absolutely within its nearest relative positioning parent */
.dead-center {
  @include nc-dead-center(absolute)
}
